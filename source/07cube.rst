07cube 魔方公式
=====================================

.. container:: sphinx-features

   .. admonition:: 🌍 魔方公式
      :class: sphinx-feature

      基本公式 ::

        上勾下回  →  勾上回下

      一字 拐弯 点 ::

        压 上勾下回 上勾下回 提

      顶角归位(眼睛朝左) ::

        上勾下提  上勾下回  下压上上回下回

      顶棱归位(好的面放后面) ::

        上回上勾上  勾上回下  回上上    (一遍不成再一遍)

   .. admonition:: 🎨 四阶五阶公式
      :class: sphinx-feature

      基本公式 ::

        上勾下(上上 勾 下下)   # 五阶十字
        上勾下      勾上回     回下(上上勾下下  勾上上回   回下) # 五阶角块

      四阶拼棱万能公式 ::

        横二回      上勾下压 下提上         横二勾

        上勾下压下提上  (翻棱公式)

      技巧 ::

        底层: 下回上   (注意回)
        顶层: 上回下

      o 特公式 ::

        上勾勾转上勾勾  上勾勾下勾勾  左下勾勾下勾勾      上勾勾下勾勾  下
                                          (五阶是三层)

      p 特公式 ::

         下下 压压 勾勾 (上上) 勾勾 压压 上上
                        回回
